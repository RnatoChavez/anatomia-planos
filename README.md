# 🦴 Planos y Ejes Anatómicos

Web interactiva para estudiar los planos anatómicos (sagital, frontal, transversal) y los ejes del cuerpo humano.

## ✨ Secciones

- **📐 Planos** — Diagrama interactivo del cuerpo con los 3 planos
- **📏 Ejes** — Tabla y diagrama de ejes anatómicos
- **🧠 Quiz** — 5 preguntas para evaluar conocimientos
- **🃏 Flashcards** — 12 tarjetas de memoria para repasar
